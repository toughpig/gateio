## 📁 项目文件说明

| 文件名 | 描述 | 适用场景 |
|--------|------|----------|
| `demo.py` | 主要演示程序，交互式菜单 | 日常使用、学习API |
| `gateio-account.py` | 账户管理功能演示 | 账户信息查询 |
| `gateio-wallet.py` | 钱包操作功能演示 | 充值提现管理 |
| `gateio-api.py` | 通用API功能演示 | API开发参考 |

## ⚡ 快速体验

如果您已经配置好环境，可以直接运行：

```bash
# 主程序（推荐）
python demo.py

# 账户管理
python gateio-account.py

# 钱包管理
python gateio-wallet.py

# 通用API
python gateio-api.py
```
